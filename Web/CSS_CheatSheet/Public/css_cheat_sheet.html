Border

La propiedad 'border' define un borde alrededor de un elemento.

Border Parts

Las propiedades 'border-left', 'border-right', 'border-top' y 'border-bottom' definen los bordes individuales de un elemento.

Border Radius

La propiedad 'border-radius' define el radio de las esquinas del borde de un elemento.

Background Image

La propiedad 'background-image' establece una imagen de fondo para un elemento.

Color

La propiedad 'color' establece el color del texto de un elemento.

Font

La propiedad 'font' es una propiedad abreviada para establecer varias propiedades de fuente en un solo lugar.

Font Family

La propiedad 'font-family' especifica la familia de fuentes para el texto de un elemento.

Font Style

La propiedad 'font-style' especifica el estilo de fuente para el texto de un elemento.

Font Weight

La propiedad 'font-weight' establece el grosor o delgadez de los caracteres en el texto.

Width and Height

Las propiedades 'width' y 'height' establecen el ancho y alto de un elemento, respectivamente.

List Style

La propiedad 'list-style' es una propiedad abreviada para establecer todas las propiedades de estilo de lista en un solo lugar.

Margin

La propiedad 'margin' establece los márgenes de un elemento.

Padding

La propiedad 'padding' establece el relleno de un elemento.

Overflow

La propiedad 'overflow' especifica qué hacer si el contenido se desborda de un elemento.

Display

La propiedad 'display' especifica el tipo de caja de visualización utilizada para un elemento.

Position

La propiedad 'position' especifica el tipo de método de posicionamiento utilizado para un elemento.

Filter

La propiedad 'filter' define efectos visuales (como desenfoque y saturación) a un elemento.
